== Description ==

This plugin adds an animated flying twitter bird in your blog. If you move the mouse over the bird, 
a "tweet this" -and displays a "follow me" button. The bird found itself as the "follow me" link on the Twitter account.
In addition to this a balloon with your latest tweet can be displayed optionally. The original script for this plugin is designed by http://floern.com, released under GNU/GPL.More Details at <a href="http://www.techlineinfo.com/wordpress-plugin-displays-animated-twitter-bird-2/">Plugin Support Page</a>


== Installation ==


1. Upload the `AnimatedTwitterBird` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter your Twitter Id by clicking the AnimatedTwitterBird under Settings menu
